# NightOps Phase 1

Base Next.js app with App Router and minimal features:
- Auth: stub login
- Map: browser geolocation display
- Camera: stub image upload
- Party: demo list

## Scripts
- `npm run dev` — start dev server
- `npm run build` — build
- `npm run start` — start production
